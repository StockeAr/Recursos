Para levantar el servidor hay dos opciones:
1) posicionados en la carpeta "server" ejecutar en una terminal el comando "npm run dev", esto usa nodemon.
2) posicionarse, dentro de la carpeta "server", en la carpeta build y ejecutar un una terminal el comando "node index.js".
Para el transpilado automatico se recomienda ejecutar en un terminal el comando "npm run build", posicionado dentro de la carpeta "server".
Para las peticiones a la base de datos se usa Postman, con la siguiente direccion " http://localhost:3000/api/prueba/ ".
